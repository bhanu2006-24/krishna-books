## ğŸš€ Getting Started

### Prerequisites
- **Node.js** (v18 or higher)
- **npm** or **yarn**

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/bhanu2006-24/krishna-books.git
   cd krishna-books
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:3000` to see the app in action!

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```
